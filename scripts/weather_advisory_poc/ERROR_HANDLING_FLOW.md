## Key Decision Points

| Failure Type | Impact | Action | Severity |
|--------------|--------|--------|----------|
| Rules JSON corrupted | **All broadcasts** | Stop, alert developer | рҹҡЁ Critical |
| Weather API timeout (1 location) | **1 group** | Skip or retry later | вҡ пёҸ Warning |
| Weather API down (all) | **All broadcasts** | Skip entire run, alert | рҹҡЁ Critical |
| Missing variety | **1 farmer** | Default to Hass | в„№пёҸ Info |
| 0 rules triggered | **1 group** | Send generic weather | вҡ пёҸ Warning |
| LLM timeout | **1 group** | Use template fallback | вҡ пёҸ Warning |
| LLM all groups fail | **All broadcasts** | Use template for all | рҹҡЁ Critical |
| WhatsApp send fails | **1 farmer** | Queue for retry | в„№пёҸ Info |
| WhatsApp API down | **All broadcasts** | Queue all for retry | рҹҡЁ Critical |
